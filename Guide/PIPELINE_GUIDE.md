# 사주명리 캐릭터 생성 시스템 — 통합 가이드

**작성일:** 2026-04-25  /  **버전:** v1.0
**대상 출력:** 1,440 캐릭터 (60일주 × 2성별 × 12월지)

---

## 1. 시스템 개요

### 1-1. 핵심 원칙
- **엑셀 = 순수 사전** — 키워드·태그·수치만 보관, 해석·조합 로직 없음
- **파이썬 = 두뇌** — 사전을 조회·조합·해석하여 결과를 생성
- 사전 수정 → 스크립트 재실행 → 모든 캐릭터에 자동 반영

### 1-2. 아키텍처 5계층

| 계층 | 번호대 | 역할 | 파일 수 |
|---|---|---|---|
| 원초사전 | 10번대 | 5오행·10천간·12지지 (가장 기초) | 3 |
| 조합사전 | 20번대 | 60갑자·10십성·12운성 (조합 결과) | 3 |
| 관계사전 | 30번대 | 합·충·형·파·해·신살 (글자 간 관계) | 3 |
| 해석사전 | 40번대 | 패턴별 해석문 (분포·조합·격국) | 4 |
| 캐릭터사전 | 50번대 | 외형·이미지 프롬프트 태그 | 4 |

### 1-3. 파일 명명 규칙
`번호_단축Prefix_이름_YYYYMMDD_HHMM.xlsx`
예) `21_GJ_60갑자_20260425_1500.xlsx`

## 3. 처리 파이프라인 (7단계)

```
[입력] → [만세력] → [기초조회] → [관계분석] → [해석] → [외형] → [프롬프트조립]
  1       2          3            4           5       6       7
```

### 1단계 — 입력 수집
- **필수:** 생년월일시 (양력/음력 구분), 성별
- **선택:** 출생지(태양시 보정), 직업, 연령

### 2단계 — 만세력 변환 (sajupy 라이브러리)
- 입력 → 사주 4주(연·월·일·시 각 천간+지지)
- 일간(日干) = 본인의 핵심 정체
- 출력 형태: 4주의 천간/지지 아이디 쌍

### 3단계 — 기초 데이터 조회 (10·20번대)

| 작업 | 조회 파일 | 추출 |
|---|---|---|
| 각 천간의 오행·음양 | 12_G_천간 | 오행ID, 음양 |
| 각 지지의 오행·지장간·생왕고 | 13_B_지지 | 오행ID, 지장간 본기, 생왕고 |
| 일주 종합 정보 | 21_GJ_60갑자 | 자좌십성, 자좌특수, 납음, 공망, 캐치프레이즈 |
| 일간 vs 다른 천간 → 십성 분포 | 22_S_십성 (시트2) | 천간별 십성 |
| 일간 vs 각 지지 → 십성 분포 | 22_S_십성 (시트3) | 지지본기별 십성 |
| 일간 vs 각 지지 → 12운성 분포 | 23_U_십이운성 (시트2) | 운성ID |

### 4단계 — 관계 분석 (30번대)

| 작업 | 조회 파일 | 검사 항목 |
|---|---|---|
| 천간끼리 합/충/극 | 31_GR_천간관계 | GR01~GR15 발동 여부 |
| 지지끼리 합/충/형/파/해/원진 | 32_BR_지지관계 | BR01~BR82 발동 여부 |
| 신살 발동 여부 | 33_SS_신살 | 각 신살별 조건 체크 |

**발동 조건 핵심**
- 천간합화: 양 천간 투간 + 인접 + 합화오행 월지 통근 + 깨는 글자 부재
- 천라지망: 진사 + 술해 4글자 모두 필요
- 인접도 우선: 일주-월주, 일주-시주가 가장 강함
- 자축합 합화: 토국 또는 수국 양설 (투간 따라)
- 자술 암합: 임정합(壬丁合)이 정설
- 사신 형: 인목 부재 시 반형
- 공망: 일주 순중 기준 우선

### 5단계 — 종합 해석 (40번대)

| 작업 | 조회 파일 | 출력 |
|---|---|---|
| 오행 분포 패턴 | 41_HE_오행균형해석 | HE 패턴 목록 |
| 십성 단일/조합 패턴 | 42_HS_십성조합해석 | HS 패턴 목록 |
| 합충 효과 종합 | 43_HC_합충효과해석 | HC 효과 목록 |
| 격국 + 용신 결정 | 44_HG_격국용신 | HG 아이디 1개 + 용신 |

**오행 분포 카운트 방법**
1. 천간 4개 각각 1점
2. 지지 4개 각각 본기 1점 + (선택) 중기 0.3점 + 여기 0.2점
3. 합화 성립 시 합화오행에 추가 점수
4. 충 성립 시 손상측 감점

### 6단계 — 캐릭터 외형 합성 (50번대)

| 작업 | 조회 파일 | 입력 | 출력 |
|---|---|---|---|
| 체형 결정 | 51_A_외형기본(체형) | 일간 오행 + 운성 | A1x |
| 얼굴형 결정 | 51_A_외형기본(얼굴형) | 일간 오행 + 강한 지지 | A2x |
| 눈매 | 51_A_외형기본(눈매) | 강한 십성 | A3x |
| 골격 | 51_A_외형기본(골격) | 오행 강약 | A4x |
| 피부톤 | 51_A_외형기본(피부톤) | 조후(월지+계절) | A5x |
| 성별·연령 | 52_AG_성별연령 | 성별, 연령대 | AG0x |
| 직업군 | 53_AJ_직업군외형 | 강한 십성 + 격국 | AJxx |
| 월지 보정 | 54_AM_월주외형보정 | 월지 | AM0x |

### 7단계 — 프롬프트 조립

**한국어 캐릭터 설명문 (LLM 입력용)**
- 일주 캐치프레이즈 (21_GJ에서 가져옴)
- 격국·용신 요약 (44_HG)
- 오행·십성 키워드 (41_HE, 42_HS)
- 자좌·신살 특이사항

**이미지 생성용 프롬프트 (스테이블디퓨전·컴피UI)**
- 51~54의 영문 프롬프트 컬럼을 결합
- 형식: `(품질) (성별·연령) (얼굴) (체형) (피부) (의상) (분위기) (계절감)`

## 4. 입출력 예시

### 입력
```
생년월일시: 1985-03-15 14:30 (양력)
성별: 여
출생지: 서울
```

### 2단계 — 사주 4주
- 연주: 을축(乙丑) — G02 + B02
- 월주: 기묘(己卯) — G06 + B04
- 일주: 갑오(甲午) — GJ31, G01 + B07
- 시주: 신미(辛未) — G08 + B08

### 3단계 — 기초 분석
- 일간: 갑(甲) 양목
- 자좌십성: S04 상관 (갑→오 본기 정火)
- 자좌특수: 왕지(도화 후보)
- 납음: 사중금(沙中金)
- 공망: 진사(辰巳)
- 오행분포: 목 1, 화 1, 토 3, 금 1, 수 0
- 십성분포: 비견 1(S01), 상관 1(S04), 정재 1(S06), 정관 1(S08)
- 12운성: 갑→축(U03 관대), 갑→묘(U05 제왕), 갑→오(U08 사), 갑→미(U09 묘)

### 4단계 — 관계 분석
- 묘+오: BR54 육파 (약)
- 지지 충 없음, 천간 합 없음, 천간 충 없음
- 신살: SS02 도화 (오는 인오술 도화)
- 천을귀인 SS04: 갑의 천을귀인은 축미 → 일지에 축, 시지에 미 모두 있음 (이중 천을귀인)

### 5단계 — 해석
- 오행: 토 과다 (HE12), 수 결핍 (HE17)
- 십성: 자좌 상관 + 정관 → 상관견관 가능성 (HS39)
- 격국: 묘월 갑목, 월지가 양인이므로 양인격(HG20)
- 용신: 양인격은 보통 관살 또는 식상으로 제어 → 시간 신금(편관) 활용

### 6단계 — 외형
- 일간 갑목 + 자좌 도화 + 운성 사(병약) → A11 수양형
- 갑목 + 묘월 → A21 장방형 얼굴
- 자좌 도화 → A34 사슴눈 또는 A31 봉안
- 갑목 음간성 약함 → A43 세골격
- 묘월(봄) → A51 청백 또는 A52 적색
- 여 30~40대 → AG06
- 양인격·상관 → AJ06(UX/UI 디자이너) 또는 AJ08(크리에이터) 등 직업 입력 따라
- 월지 묘 → AM04 봄 한창

### 7단계 — 프롬프트

**한국어 설명**
> 갑오 일주의 30대 후반 여성. 봄날 만개한 정원의 화초처럼 또렷하고 곱지만, 양인격 특유의 단단한 자존심을 지님. 자좌가 상관(도화)이라 표현력과 매력이 뛰어나며, 천을귀인 두 자리(축·미)로 결정적 인덕도 보유. 다만 정관(시간 신금)이 상관에 노출되어 있어 구설·관재에 주의 필요. 토 과다·수 결핍으로 통찰보다는 안정·축적 성향이 강하니, 의도적으로 깊이 파고드는 시간 확보가 보완책.

## 6. 검증 규칙 (정설 기반)

### 6-1. 사주 4주 결정
- sajupy 결과를 기본 신뢰
- 자시(子時) 처리: 23:30 이후는 다음날 일주(조자시) 또는 당일 야자시 — sajupy 기본값 따름
- 절기 기준 월주 (입춘 = 인월 시작)
- 태양시 보정 옵션 (서울 기준 약 −32분)

### 6-2. 합·충 발동 조건

| 항목 | 조건 |
|---|---|
| 천간합화 | 양 천간 투간 + 인접 + 합화오행이 월지 통근 + 깨는 글자 부재 |
| 육합 | 인접 우선, 멀면 약함 |
| 삼합 | 三字 동시 + 왕지(子午卯酉) 中心 (왕지 빠지면 약) |
| 방합 | 三字 동시 (계절적 흐름) |
| 천라지망 | 진사 + 술해 4글자 모두 필요 |

### 6-3. 학파별 차이 메모
- 음간 양인: 학파별로 인정 여부 다름 — 본 시스템은 표시만
- 오행 가축 매핑: 영추 vs 소문 차이 — 본 시스템은 영추 기준
- 십이운성 매핑: 음간 역행 적용 — 정설 표 사용
- 자축합 합화: 토국·수국 양설 (투간 따라 결정)
- 자술 암합: 임정합(壬丁合)이 정설 (계정합 아님)
- 사신: 합과 형 동시 (반형으로 약함)
- 공망: 일주 순중 기준 우선

### 6-4. 인접도 원칙
- 일-월: 매우 강
- 일-시: 매우 강
- 월-년: 강
- 년-시: 약 (거리 멀음)
- 합충 모두 인접도 따라 강도 가중

### 6-5. 캐릭터 생성 시 제외 사항
- 천간합화 → 캐릭터 생성 로직에서는 제외 (조건 엄격)
- 같은 오행 과다 → 부조화로 해석 (단순 강화 아님)
- 트리비얼 조합(년-시 약한 충 등) → 각주만 표시 또는 생략

## 8. 자주 발생하는 함정

1. **자좌십성을 오행 분포에 중복 카운트하지 말 것** — 일지 본기는 한 번만 카운트
2. **합화 성립 후에도 원래 오행 점수는 유지** — 단, 합화오행에 추가 점수 부여
3. **공망 위치는 효과 약화** — 해당 글자의 영향력 감소
4. **편관(칠살)과 정관 동시 존재 시(관살혼잡)** — 하나로 정리되는 합 또는 제어 글자 확인
5. **상관견관**은 정관이 손상되어 명예 위태 — 인성으로 보호 필요
6. **신약한데 식상·재·관 과다**면 신왕화 보완(인성·비겁) 우선
7. **여명에서 관성**은 남편 — 충/형 시 결혼·이성 관련 변동 해석
8. **양인격·건록격은 외격 아닌 정격의 변형** — 격국 분류 시 주의
