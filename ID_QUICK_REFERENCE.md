## 핵심 발동 조건 요약

| 항목 | 조건 |
|---|---|
| **천간합화** | 양 천간 투간 + 인접 + 합화오행 월지 통근 + 깨는 글자 부재 |
| **육합** | 인접 우선 |
| **삼합** | 三字 동시 + 왕지(子午卯酉) 中心 |
| **천라지망** | 진사 + 술해 4글자 모두 필요 |
| **공망** | 일주 순중 기준 |
| **양인** | 양간만 (갑→묘, 병/무→오, 경→유, 임→자) |
| **천을귀인** | 갑·무·경 → 축·미 / 을·기 → 자·신 / 병·정 → 해·유 / 임·계 → 사·묘 / 신 → 인·오 |

## 인접도 가중

```
  연  ─── 월  ─── 일  ─── 시
  년주    월주    일주    시주
   │      │      │      │
   ├──약──┤      ├──강──┤
          ├──강──┤
   ├──────────약──────────┤
```

- **일-월·일-시 = 매우 강**
- **월-년 = 강**
- **년-시 = 약**

## 처리 흐름 한 줄 요약

```
입력 → 만세력(sajupy) → 사주 4주 → 기초조회(10·20번대)
     → 관계분석(30번대) → 해석(40번대) → 외형(50번대)
     → 한국어 설명 + 이미지 생성 프롬프트
```
